Bootstrap Buttons

Bootstrap provides an easy way to get a range of buttons with diiferent styles.

 

 

Below is the code for the default button style -

<button type="button" class="btn btn-default">Default</button>

 

Button Styles

The default style can be changed using the class codes below.

Just add the style class that you want to the button class to change the size of your button -

<button type="button" class="btn btn-default">OK</button>

 

Roll the mouse over the buttons and you will see them react to the mouse hover.

 

Class Button Code
.btn-default <button type="button" class="btn btn-default"> Default</button>
.btn-danger <button type="button" class="btn btn-danger"> Danger</button>
.btn-info <button type="button" class="btn btn-info"> Info</button>
.btn-link <button type="button" class="btn btn-link"> Link</button>
.btn-primary <button type="button" class="btn btn-primary"> Primary</button>
.btn-success <button type="button" class="btn btn-success"> Success</button>
.btn-warning <button type="button" class="btn btn-warning"> Warning</button>

 

 

Button Size

There are four classes for setting the size of Bootstrap's buttons - large, medium, small and extra small.  

Just add the size class to the style class to change the size of your button -

<button type="button" class="btn btn-info btn-lg">Info</button>

 

Large - .btn-lg creates a Large button

 

Medium - .btn-md creates a Medium button

 

Small - .btn-sm creates a Small button

 

Extra Small - .btn-xs creates a Extra Small button

 

 

Block Level Button

A block level button expands it's width to fill the parent object.

Just add the block class to the style class to change it to a block level button

<button type="button" class="btn btn-info btn-block">Block Button</button>

 

 

You can add the button size class as well

<button type="button" class="btn btn-info btn-block btn-lg">Block Button</button>

 

 

 

Button States

Buttons can have three states - normal, active and disabled.

Normal is the default. The class .active makes a button look as though pressed, and the .disabled class makes a button unclickable.

just add active or disabled to the class to make a button active or disabled.

 

<button type="button" class="btn btn-info">Normal Button</button>

<button type="button" class="btn btn-info active">Active Button</button>

<button type="button" class="btn btn-info disabled">Disabled Button</button>

 

 

 

Button Classes for Hyperlinks and Form Buttons

Button classes can also be used on the <a> and <input> elements to help format hyperlinks and form buttons.

 

Hyperlink example - <a href="#" class="btn btn-primary" role="button">Refresh</a> - Refresh

Input example  - <input type="submit" class="btn btn-info" value="Submit"> -

 

Note - the # character used in the href="#" acts like an address to a Web page for this demonstration. It has the effect of refreshing the page.
