Bootstrap Well

The .well class creates a box type element with padding, rounded corners and a grey background. All you need is a <div class="well"> to create a well that stretches across the entire screen.

 

Basic Well

 

The well below has been placed inside a .container class layer.

 

Basic Well in a container class

 

Well Size

You can change the size of the well by using the .well-sm class for small wells or .well-lg class for large wells

Normal (basic) Well

Well Colour

You can change the background colour of the well by altering the boootstrap css using your css file. Add this code .well {background-color:#FF7519;} to your CSS or code it as inline css in the <div>.

All entries for the .well class in bootstrap.css -

  .well {
     min-height: 20px;
     padding: 19px;
     margin-bottom: 20px;
     background-color: #f5f5f5;
     border: 1px solid #e3e3e3;
     border-radius: 4px;
     -webkit-box-shadow: inset 0 1px 1px rgba(0, 0, 0, .05);
     box-shadow: inset 0 1px 1px rgba(0, 0, 0, .05);
   }

 

   .well blockquote {
     border-color: #ddd;
     border-color: rgba(0, 0, 0, .15);
   }

 

   .well-lg {
     padding: 24px;
     border-radius: 6px;
   }

 

   .well-sm {
     padding: 9px;
     border-radius: 3px;
  }
