Bootstrap Grid

 

Bootstrap works on a grid system. Using the grid you can divide the page up into a maximum of 12 columns. You can also group columns together to split the grid into other multiples of twelve. For example, you could have three wider columns of 4 (3 x 4=12) or two even wider colums of 6 (2 x 6=12) and so on. Most of these permutations are shown in the coloured grid below. Like other elements in Bootstrap the grid is responsive and will react to different device sizes.
